## Live Link
 [https://my-app-alpha-lyart.vercel.app/](https://my-app-alpha-lyart.vercel.app/)

## Technologies
 
 - Vue
 - TailwindCSS
 - Vercel

## Project Setup

1. Run dependencies using npm install
2. Run server using npm run dev
3. compile and minify for production using npm run build
